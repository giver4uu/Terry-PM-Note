## 🎯 고객사별 니즈 분석

### 1. 페이타랩 (46개 질문) - 가장 활발

**주요 니즈:**
- 📅 **주간 채용 현황 보고서**: 매주 인터뷰 단계별 인원/포지션 정리
- 🔍 **특정 지원자 검색**: 역량 기반 지원자 서칭 (AI, 데이터 분석 등)
- 📊 **유입경로별 통계**: 포지션 × 유입경로 × 기간별 다차원 분석

**페인포인트:**
- ❌ AI 응답 정확성 불신 ("데이터가 하나도 맞지 않아")
- ❌ 기간 해석 오류 ("아니 나는 2026년 1월 내를 말하는 거야!")
- ❌ 용어 정의 혼란 ("합격자 = 서류 통과자? 최종 합격자?")
- ❌ 지원자 식별 어려움 ("지원자 이름이 숫자라서 알 수가 없어")

**기회 영역:**
> 📌 **정기 보고 템플릿 자동화**: 매주 동일 형식 요청 → 스케줄링/템플릿 기능
> 📌 **역량 기반 지원자 추천**: AI 효율화, 자동화 역량 보유자 검색 기능

### 2. 게임듀오 (21개 질문)

**주요 니즈:**
- 📅 **주차별 진행현황 보고**: 4개 공고(QA, 아트, 사업개발, 중국어) 정기 분석
- 📈 **탈락 패턴 분석**: 불합격 86명의 단계별 탈락 사유
- 🔎 **유입경로 세분화**: 서칭, 현장실습 등 채널별 분석

**페인포인트:**
- ❌ 응답 없음 현상 ("왜 결과에 아무말이 없니")
- ❌ 과거 데이터 혼재 ("합격 3명은 2년전 데이터라 무관해")

**기회 영역:**
> 📌 **공고별 리포트 템플릿**: 특정 4개 공고 기준 주간 보고 자동화
> 📌 **기간 필터 강화**: 최근 N개월 데이터만 분석하는 옵션 명확화

## 🚀 제품 개선 제안

### 즉시 개선 (Quick Win)

1. **용어 가이드 표시**: "합격자"를 선택할 때 "서류 합격 / 면접 합격 / 최종 합격" 중 선택하도록 UI 개선
2. **기간 확인 프롬프트**: 분석 전 기간 범위를 명시적으로 보여주고 확인 받기
3. **템플릿 저장**: 자주 사용하는 쿼리를 템플릿으로 저장하고 재사용

### 중기 개선 (3-6개월)

4. **정기 리포트 자동화**: 매주 특정 요일에 자동으로 보고서 생성/발송
5. **탈락 사유 자동 분류**: 평가 코멘트 NLP 분석으로 패턴 도출
6. **리드타임 대시보드**: 전형 단계별 병목 구간 시각화

### 장기 개선 (6개월+)

7. **역량 기반 추천**: 특정 역량 키워드로 적합 지원자 자동 추천
8. **예측 분석**: 지원자 트렌드 예측, 채용 성과 예측
